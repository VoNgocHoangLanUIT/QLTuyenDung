# 🧠 Recruitment Management Website

**Duration:** Feb 2025 – Present  
**Client:** TalentHub – Human Resources Service Provider  
**Team:** University Capstone Project  
**Technologies:** Node.js / ReactJS / PostgreSQL / LinkedIn API / SMS Gateway / AI Matching Engine

---

## 🚀 Project Overview

A full-featured recruitment platform designed for **TalentHub**, a company specializing in hiring services for businesses. The system facilitates job posting, application management, interview scheduling, and candidate evaluation — offering tailored tools for recruiters, HR staff, and job seekers.

With over **500+ registered companies** and **5000+ applicant profiles**, the system scales to meet enterprise-grade demands, while supporting modern features such as AI-powered recommendations and third-party API integrations.

---

## 🧩 Key Features

### 🎯 For Recruiters:
- Create and publish job posts by industry category.
- Manage multi-branch job listings (e.g., HCMC, Hanoi, Danang).
- Filter and review candidate profiles.
- Schedule interviews and online tests (multiple rounds supported).
- Create short quizzes (5–10 questions) or in-depth assessments.
- Evaluate candidates and update results after each interview round.
- Manage roles and permissions for HR staff securely.

### 🧑‍💼 For Applicants:
- Create and manage job profiles with resume uploads.
- Browse job listings by location, industry, or contract type (full-time, part-time, freelance, short-term).
- Apply online and track the progress of their applications.
- Receive notifications about interview/test schedules.
- Delete personal profiles if no longer needed.

### 🔧 System Features:
- **AI Integration**: Recommend suitable jobs to candidates based on profile data.
- **LinkedIn Integration**: Auto-import applicant data using LinkedIn API.
- **Email Alerts**: Notify users about new matching jobs or updates.
- **SMS Reminders**: Send reminders prior to interviews or tests to reduce no-shows.
- **Responsive Design**: Cross-device compatibility with an intuitive UI.
- **Role-Based Access Control**: Secure data by defining access levels for each user type.

## 🧱 Architecture & Development Environment

- Modular development with fixed review milestones.
- Each module developed following standard software engineering practices:
  - **UML Modeling**: Use Case, Activity, Sequence, and Class Diagrams
  - **Software Architecture**: MVC / RESTful API structure
  - **Version Control**: Git & GitHub

> Initial focus is placed on core features like job posting and profile management. Advanced modules like AI matching and chatbot support are implemented in later stages.

---

## 🛡️ Security & Data Protection

- Secure authentication and session management.
- Role-based access and field-level restrictions for data privacy.
- Sanitization of user inputs to prevent XSS and SQL injection.
- Encrypted storage for sensitive information (e.g., user credentials).
